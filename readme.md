# UnityUtil
一些Unity常用工具合集，直接上代码

# 基础
基于Unity2020 Urp管线；因为URP和HDRP已稳定，遂不在支持多版本，只使用支持手游的URP

引用库
1. Cinemachine  -   因为经常想做运镜和镜头语言，所以就顺手引入
2. Universal RP
